# 🔐 Password Strength Checker

This is a Python project that checks the strength of a password using conditional statements and string handling.

## 🚀 Features
- Checks minimum length (8 characters)
- Detects:
  - Uppercase letters
  - Lowercase letters
  - Digits
  - Special characters
- Classifies password:
  - Weak
  - Medium
  - Strong

## 🧠 Concepts Used
- Strings
- Conditional Statements (if-elif-else)
- Loops
- Boolean Flags

## ▶️ How to Run

```bash
python password_checker.py
